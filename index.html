      How To

    • To create a task? Fill in the task title, and a task description if you'd like, and add options then click "Create Task".
    • To edit a task? Click on the task more options button on the task you want to edit, then click "Edit", make your changes, and click "Save".
    • To move a task to the trash? Click on the task more options button on the task you want to move to the trash, then click "Move to trash".

      • FAQ

      • Is Munetios Tasks free? Yes, Munetios Tasks is free to use.
      • Does it work offline? Yes, Munetios Tasks is offline-capable. Your tasks are stored locally in your browser.
      • How are my tasks and categories secured? All tasks and categories are encrypted using AES-GCM before being stored in your browser's IndexedDB.
      • Can I organize tasks by category? Yes, you can create custom categories and assign tasks to them.
      • How do I restore a task from Trash? Go to the Trash page, click the more options button on the task, and select "Restore".
      • How do I permanently delete a task? In the Trash page, click the more options button and select "Delete Permanently".
      • Can I print my tasks? Yes, you can print individual tasks or all tasks using the "Print" options.
      • Is my data synced across devices? No, currently your data is stored only on the device/browser you use.
      • How do I mark a task as completed? Click the more options button on a task and select "Mark as Complete", or check all options/subtasks for auto-completion.
      • Can I add images to tasks? Yes, click the image icon when creating a task to attach an image.
